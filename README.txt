DESCRIPCIÓN GENERAL
-------------------
Este proyecto desarrolla un sistema de transporte utilizando Oracle Database.
Incluye la creación, diseño e implementación de la base de datos a través de las
etapas de modelado Entidad–Relación, modelo lógico y modelo físico.

CONTENIDO DEL PROYECTO
-----------------------
1. Diagramas:
   - Diagrama Entidad–Relación (E-R)
   - Diagrama Lógico
   - Diagrama Físico

2. Script SQL:
   - Archivo principal de creación de tablas (compatible con Oracle)
   - Definición de llaves primarias, foráneas, secuencias e índices

3. Documentación:
   - Manual técnico
   - Manual de usuario
   - Diccionario de datos

TECNOLOGÍAS UTILIZADAS
-----------------------
- Oracle Database 19c
- PowerDesigner (para modelado)
- GitHub (para control de versiones y respaldo)

EJECUCIÓN
----------
1. Abrir Oracle SQL Developer o una consola SQL compatible.
2. Ejecutar el script CREATE_TABLES.sql en el esquema correspondiente.
3. Verificar la creación de las tablas mediante:
   SELECT table_name FROM user_tables;

OBSERVACIONES
--------------
El proyecto cumple con las buenas prácticas de diseño relacional y 
garantiza integridad referencial, consistencia de datos y documentación completa.
